# Module units

Type-safe units of measure for length, area, and rotation, inspired by `kotlin.time.Duration`.

# Package io.github.mapvina.spatialk.units

Core value classes for physical and angular measurements (Length, Area, Rotation, Bearing) with
predefined unit definitions (SI, Metric, Imperial, DMS).

# Package io.github.mapvina.spatialk.units.extensions

Kotlin DSL extensions for ergonomic unit construction and conversion, including extension properties
on numeric types (e.g., `5.meters`, `45.degrees`) and utilities for collections and trigonometry.
